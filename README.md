# Smart Logistics Optimization Platform

A production-grade, microservices-based logistics platform connecting Truck Drivers, Fleet Owners, Businesses, and Admins.

## Tech Stack
- **Frontend**: React.js (JavaScript), Vite, React Router, Axios, React Query, TailwindCSS, Redux Toolkit, React Hook Form, Zod, Framer Motion
- **Backend**: Java 21, Spring Boot 3, Spring Cloud, Spring Security, Spring Data JPA, MapStruct, Lombok, OpenFeign
- **Database**: PostgreSQL
- **Caching**: Redis
- **Messaging**: Apache Kafka
- **Storage**: Local Storage (ready for AWS S3)
- **Architecture**: Microservices, API Gateway, Eureka Service Discovery, Config Server

## Project Structure
- `backend/`: Spring Boot Microservices
- `frontend/`: React Frontend Applications
- `docs/`: Architecture and development guidelines
- `scripts/`: Development and deployment scripts
